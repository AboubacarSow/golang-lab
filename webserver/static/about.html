About Golang Lab

Small examples showing how to serve static files and simple endpoints using Go's standard library.
v1 · Demo

Purpose

This repository is a lightweight playground for learning Go's built-in HTTP server and for experimenting with static file serving, routing, and tiny APIs without external dependencies.

Quick start

Place files in the /static folder (this page is /static/about.html), then run your server and open http://localhost:8080/.

// Example main.go
package main

import (
	"log"
	"net/http"
)

func main(){
	fs := http.FileServer(http.Dir("./static"))
	http.Handle("/", fs)
	log.Println("Listening on :8080")
	log.Fatal(http.ListenAndServe(":8080", nil))
}

What you'll find here

  • Simple static HTML pages (index, about).
  • Ready-to-run Go server examples using only the standard library.
  • A small base for adding JSON endpoints, templates, or middleware.

Extending the project

To add routes or JSON APIs, replace the default handler registration with a router or add custom http.HandleFunc handlers. The standard library is intentionally simple and great for learning.
