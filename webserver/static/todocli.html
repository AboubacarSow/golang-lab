todocli — CLI Todo Manager

A command-line todo app demonstrating idiomatic Go, JSON persistence, and pipe-friendly input handling.

Overview

The todocli mini-project provides a small todo manager with add, list, complete, and delete operations. It stores tasks in todos.json and uses standard library packages only.

Features

  • Add todos with command flags
  • List active and completed todos
  • Mark tasks complete
  • Delete items by index
  • Support for piped todo input

Concepts

  • CLI flag handling
  • JSON encoding and decoding
  • File I/O and persistence
  • Slice manipulation and pointer semantics

Run locally

cd ../todocli
go run ./cmd/todo -list
go run ./cmd/todo -add "Buy groceries"
go run ./cmd/todo -complete 1
go run ./cmd/todo -delete 1

Pipe input:

echo "Buy groceries" | go run ./cmd/todo -add

Structure

  • cmd/todo/ — CLI entry point and command parsing
  • todo.go — todo model and persistence logic
  • todos.json — storage file for tasks

Why it matters

todocli is a compact example of building CLI tooling with Go and managing state in a file, which makes it a nice first project for learning standard library patterns.
