// server.go
package main

import (
	"net/http"
	"log"
)

func main(){
	http.Handle("/", http.FileServer(http.Dir("./static")))
	port := ":8080"
	fmt.Println("Server starts running on port" + port)
	if err := http.ListenAndServe(port, nil); err != nil {
		log.Fatal(err)
	}
}

Then run: go run main.go

About this project

This workspace is a small playground for learning Go's standard library HTTP server and static file serving. Feel free to modify the HTML, add styles, or expand the server with JSON APIs.

Built with simplicity and clarity in mind — suitable for demos and quick experiments.
